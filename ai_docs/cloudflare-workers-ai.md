# Cloudflare Workers AI Documentation

## Overview

Workers AI enables users to execute machine learning models serverless through GPU infrastructure on Cloudflare's network, accessible from Workers, Pages, or the Cloudflare API.

## Key Capabilities

The platform provides:
- Access to "50+ open-source models" through the model catalog
- "Serverless, pay-for-what-you-use" cost structure
- Integration with a comprehensive developer ecosystem featuring AI Gateway, Vectorize, and Workers

## Availability

The service launches on both Free and Paid plan tiers and has achieved General Availability status.

## Support & Feedback

Users can report issues or provide feedback through the #workers-ai Discord channel. Wrangler-specific problems should be directed to the Wrangler GitHub repository.

## Custom Deployment

Organizations with specialized requirements, such as proprietary model deployments or expanded usage limits, may submit inquiries via the Custom Requirements Form for tailored solutions.

## Functional Features

**Models**: The platform includes curated open-source models supporting image classification, text generation, object detection, and additional machine learning tasks.

## Related Ecosystem Products

- **AI Gateway**: Provides observability and management tools
- **Vectorize**: Vector database for semantic search capabilities
- **Workers**: Serverless application framework
- **Pages**: Full-stack application deployment
- **R2**: Object storage infrastructure
- **D1**: SQL database service
- **Durable Objects**: Distributed coordination system
- **KV**: Global key-value storage
